# Tech Arsenal

Minecraft 1.20.1 (Forge) 向けテクノロジーMOD集。近未来的な監視・攻撃テクノロジーをMinecraftに追加します。

## 収録予定のテクノロジー

- **衛星カメラ (Satellite Camera)** — 上空からワールドを俯瞰できる衛星視点カメラ
- **監視カメラ (Security Camera)** — 設置型の監視カメラ。遠隔地からリアルタイムで映像を確認
- **誘導ミサイル (Guided Missile)** — ターゲットを追尾する誘導ミサイル
- **戦闘ドローン (Combat Drone)** — 自律行動する戦闘用ドローン

## 開発環境

| 項目 | バージョン |
|---|---|
| Minecraft | 1.20.1 |
| Forge | 47.4.10 |
| Java | 17 |
| Mod ID | `techarsenal` |

## ビルド方法

```
.\gradlew.bat build
```

成果物は `build/libs/` に出力されます。

## クライアント起動(開発用)

```
.\gradlew.bat runClient
```

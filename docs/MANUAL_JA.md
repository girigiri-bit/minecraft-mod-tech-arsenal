# Tech Arsenal プレイヤーズマニュアル

Tech Arsenal Mod (Forge 1.20.1) の全プレイヤー向け機能を解説する取扱説明書です。
細かい数値の正はすべて `docs/WEAPONS.md` にあります。本書ではプレイに必要な数値
（射程・クールダウン・弾薬数など）だけを抜粋して掲載しています。

ゲーム内でもこの内容を要約した本『取扱説明書』(`field_manual`) を読むことができます
（クリエイティブタブ「テックアーセナル」の先頭にあります）。

## 1. はじめに

Tech Arsenal は、監視カメラ・武器・ドローンやタレットなどの設置兵器・戦車やヘリなどの
ビークル・顔認証セキュリティシステムを追加するテクノロジーMODです。

- **サバイバルモード**では、実弾武器やビークル搭載兵器は自分のインベントリから弾薬を
  消費します。弾薬が尽きると発射できません（マシンガンは弾切れで自動的に連射を止めます）。
- **クリエイティブモード**では弾薬は一切消費されません。
- レーザーガンとビームサーベルはエネルギー式のため、モードに関わらず弾薬を必要としません。
- 困ったときはいつでもゲーム内の取扱説明書アイテム `field_manual`（和名：取扱説明書）を
  右クリックすると、この内容を要約した本を読み返せます。

## 2. カメラ・監視システム

### 監視カメラ (`security_camera`)

壁や床に設置する小型カメラブロックです。設置すると自動的に **CAM番号**（CAM-1、CAM-2…）が
割り振られます。番号はディメンションごとに個別に管理され、ワールドに保存されるため
リログしても変わりません。

### カメラモニター (`camera_monitor`)

携帯用の視聴端末です。

- 監視カメラを**右クリック**するとそのカメラにリンクします。
- 何もない空中を**右クリック**すると、リンクしたカメラの視点に切り替わります
  （カメラから**64ブロック以内**にいる必要があります）。
- **スニーク**すると通常の視点に戻ります。

### 衛星リモコン (`satellite_remote`)

**使用**すると、自分の真上**80ブロック**の上空から真下を見下ろす俯瞰視点に切り替わります。
**スニーク**で解除します。設置物や弾薬は不要です。

### モニターブロック (`monitor`)

壁掛けの表示画面です。同じ向きで隣接して設置すると、最大 **5×5** まで1枚の大画面として
連結します（連結ルールはモニター同士の向きが揃っていることです）。

- **右クリック**でフィードを切り替えます: `OFF → SAT（衛星視点） → CAM-1 → CAM-2 → …`
- **スニーク右クリック**で逆順に切り替わります。
- 画面には**リアルタイム映像**と、現在の表示元（`SAT` / `CAM-n` / `NO SIGNAL` / `OFF`）の
  IDラベルが表示されます。

## 3. 武器

携行武器はすべて片手持ちです。実弾武器は弾薬（第7章参照）を消費します。

| 武器 | 弾種 | 特徴 |
|---|---|---|
| ライフル (`rifle`) | 弾丸 | 高精度、単発。約2.5発/秒 |
| マシンガン (`machine_gun`) | 弾丸 | ホールドで連射（10発/秒）、拡散大。弾切れで自動停止 |
| グレネードランチャー (`grenade_launcher`) | グレネード弾 | 山なり弾道。着弾で半径2.5m爆発、**ブロック破壊なし** |
| ロケットランチャー (`rocket_launcher`) | ロケット弾 | 直進弾。着弾で半径3.5m爆発、携行版は**ブロック破壊なし** |
| レーザーガン (`laser_gun`) | 不要（エネルギー式） | 撃った瞬間に着弾するヒットスキャン。約2発/秒、射程64m |
| ビームサーベル (`beam_saber`) | 不要（近接） | 下記参照 |
| 火炎放射器 (`flamethrower`) | ブレイズパウダー | ホールドで円錐状に放射。射程8m |
| ミサイルランチャー (`missile_launcher`) | 誘導ミサイル | 下記参照 |

### ロケットランチャーとロケットポッドの違い

携行のロケットランチャーはブロックを破壊しません。ただし、**攻撃ヘリに搭載されたロケット
ポッドは地形を破壊します**（クレーター小）。詳しくは第6章「乗り物」を参照してください。

### ビームサーベル (`beam_saber`)

近接専用武器で、ダメージだけでなく**防御手段**にもなります。

- **通常のスイング中**は、正面〜側面（視線の前方110°の範囲）2.5m以内に飛んできた弾や
  レーザーを**弾き返します**。弾き返した弾は自分の攻撃扱いになります。
- ヒットスキャン式のレーザーガンの一撃も弾くことができ、この場合は**撃った相手に
  ダメージが反射**されます。
- **Rキー**で3連コンボ技を発動します: `袈裟斬り → 水平斬り → 全周回転斬り＋ノックバック`。
  技と技の間隔は約0.35秒、コンボ全体のクールダウンは**5秒**です。

### 火炎放射器 (`flamethrower`)

燃料は**ブレイズパウダー**です。使用を開始した瞬間と、そこから**2秒ごと**に1個ずつ消費します
（クリエイティブは無消費）。

### ミサイルランチャー＋誘導ミサイル (`missile_launcher` / `guided_missile`)

**使用**すると、視線コーン内・**48ブロック以内**の敵をロックオンして誘導ミサイルを発射します。
ミサイルはターゲットを自動追尾します。もし**レーザー指示器**（第4章）でマークした対象が
いれば、そちらが最優先でロックされます。

## 4. サポート装備

### レーザー指示器 (`laser_designator`)

味方の攻撃を支援するマーキング装備です。

- **右クリック**で、視線の先**60ブロック以内**の生物をマーキングします（有効時間**60秒**、
  対象は発光します）。
- マークした対象は、誘導ミサイルの**最優先ロック対象**になります。この場合ミサイル側の
  視線ロックは不要で、**128ブロックまで**追尾を継続します。
- **スニーク右クリック**でマーキングを解除します。

### ドローン強化モジュール (`drone_upgrade_damage` / `drone_upgrade_armor`)

ドローン（第5章）に対して**右クリック**で使用すると、性能を強化するモジュールです。
それぞれ**最大Lv2**まで強化でき、適用した瞬間にドローンは全回復します。

| モジュール | 効果 |
|---|---|
| 攻撃モジュール (`drone_upgrade_damage`) | ボルト威力: 5 → 8 → 11 |
| 装甲モジュール (`drone_upgrade_armor`) | 最大HP: 20 → 30 → 40 |

## 5. 設置型・タレット・地雷

### 地雷 (`landmine`)

設置すると、触れたすべての **LivingEntity**（プレイヤー・Mob問わず、味方も含む）に反応して
起爆します。爆発半径は約3.0mですが、**ブロックは破壊しません**。

### 防衛タレット (`defense_turret`)

設置型の自動迎撃兵器です。射程**24ブロック**以内のモンスターを自動的に索敵・攻撃します
（ダメージ8/発、発射レート約1.33発/秒）。HPは40です。

- 回収するには、**素手でスニーク右クリック**します（アイテムに戻ります）。

### ドローン (`drone`)

召喚するとオーナーに追従して飛行する小型の護衛ユニットです。射程**16ブロック**以内の
モンスターを自動索敵し、`drone_bolt` で攻撃します。HPは初期20（強化モジュールで最大40まで
増加）。

- 回収するには、**素手でスニーク右クリック**します。強化レベルは回収したアイテムにも
  引き継がれます。

## 6. 乗り物

### 攻撃ヘリ「アパッチ」(`attack_helicopter`)

HP60の飛行ビークルです。

- **W** — 視線方向に飛行します（見上げる/見下ろすことで昇降します）。
- **S** — 後退します。
- **スニーク** — 降機します。
- 落下ダメージは無効です。
- **R** — 機体前方のロケットポッドを発射します（直撃24ダメージ、**地形破壊あり**、
  左右のポッドが交互に発射、クールダウン1秒）。弾薬は搭乗者インベントリの `rocket` を消費します。

### 戦車 (`tank`)

HP100の陸上ビークルです。

- **WASD** — 走行します（視線の向きが車体の正面になります）。1段（1ブロック）までの
  段差を乗り越えられます。
- ノックバックを**完全無効化**します。
- **R** — 主砲を発射します（直撃30ダメージ、**地形破壊あり**、クールダウン3秒）。
  弾薬は搭乗者インベントリの `shell` を消費します。

### 乗り物共通仕様

- 搭乗中でも手持ちの武器（ライフルなど）を通常通り使用できます。
- 未展開の状態はただのアイテムとして持ち運べます。
- 回収するには、**素手でスニーク右クリック**します。

## 7. 弾薬とクラフト

サバイバルでは、対応する武器を使用するたびにインベントリから自動で弾薬が消費されます
（クリエイティブは無消費）。レーザーガンとビームサーベルは弾薬不要です。

| アイテム | 使用武器 | クラフト |
|---|---|---|
| `bullet`（弾丸） | ライフル / マシンガン | 鉄インゴット + 火薬 → 8個 |
| `grenade`（グレネード弾） | グレネードランチャー | 鉄インゴット + 火薬×2 → 2個 |
| `rocket`（ロケット弾） | ロケットランチャー / ヘリのロケットポッド | 鉄インゴット + 火薬×3 → 2個 |
| `shell`（戦車砲弾） | 戦車の主砲 | 鉄インゴット×2 + 火薬×2 → 2個 |
| ブレイズパウダー（バニラ） | 火炎放射器の燃料 | — |

## 8. セキュリティシステム

### 顔認証スキャナー (`face_scanner`)

正面**3ブロック**のゾーンに立った人物を**5tickごと**にスキャンする設置ブロックです。

- 設置したプレイヤーは**自動的にオーナー登録**されます。
- オーナーが**素手で右クリック**すると**登録モード**（**30秒**）が始まり、その間にゾーンへ
  立った人を新たに登録できます。
- オーナーが**スニーク右クリック**すると、登録リストを**全消去**します。
- **登録済みの人物が近づくと**: 緑のパーティクル＋チャイム音が鳴り、半径**4ブロック**以内の
  ドアが自動で開きます。ゾーンから離れて約**1.5秒後**に自動で閉まります。
- **未登録の人物が近づくと**: 赤いパーティクル＋ブザー音で拒否されます（再発報まで**2秒**の
  クールダウンがあります）。

### 認証モニター (`auth_monitor`)

顔認証スキャナーから**4ブロック以内**に設置すると自動でリンクする表示板です。

- 待機中は**青**、認証OKで**緑のチェックマーク**、認証NGで**赤の×**に表示が切り替わります。

### セキュリティドア (`security_door`) と鍵 (`door_key`)

- **未施錠**の状態では、誰でも開閉できます。
- 未施錠のドアを**鍵で右クリック**すると**施錠**されます。このとき鍵に固有のIDが
  刻印されます（初めて施錠したときだけ）。
- 施錠後は、**一致する鍵**を持っている人だけが開閉できます。鍵が合わない場合は
  赤いエフェクト＋ブザー、鍵を持っていない場合は「施錠中」と表示されます。
- 一致する鍵で**スニーク右クリック**すると**解錠**できます。
- **顔認証スキャナーからは、施錠状態に関わらず登録者がドアを開けられます**
  （第8章前半の自動ドア機能はロック状態を無視します）。
- ドアを破壊すると施錠記録も消去されます。
- **レッドストーンでは開閉できません**（施錠中でなくても）。
- 鍵のツールチップには「未紐付け」または「紐付け済み」（アクア色の文字）が表示され、
  どの鍵がどのドアに使われたか区別しやすくなっています。
